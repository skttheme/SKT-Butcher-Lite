== Description ==

SKT Butcher Lite. WordPress theme can be used for Butcher Shops, Meat Stores, Steakhouses, Poultry Farms, Meat Delivery Services, Food Supply Businesses and Grocery Stores. It is responsive mobile friendly and WooCommerce compatible. Has call to action and is SEO friendly. Easy to edit with Elementor.
